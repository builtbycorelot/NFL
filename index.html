NodeForm Language (NFL)

NodeForm unifies computation and policy through a semantics-first graph syntax. The repository distills the core DSL where nodes and edges express relationships that can compile to diverse runtimes such as WASM, CUDA, and neuromorphic hardware.

Standards and Interoperability

NFL aims to align with emerging W3C efforts around describing things and data models so that schemas can tie out to major formats across the web.

Visualization

Graphs expressed in NFL can be rendered with D3.js for interactive exploration of nodes and their rel^n connections.

Process and Communications

The language provides a way to model processes and standard communication patterns using simple node and edge constructs.

Example Snippet

pack meta.core

node Party { ... }
edge submits : Party -> PermitApplication

CLI tooling validates graphs and can export an OpenAPI document:

$ python -m cli.nfl_cli examples/simple.json --export-openapi graph.openapi.json
